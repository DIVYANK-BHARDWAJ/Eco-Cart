## 📖 Project Overview

**Eco Cart** is an innovative and user-friendly sustainable shopping platform designed to help individuals make environmentally responsible choices in their everyday lives. The project aims to simplify the transition from conventional household products to eco-friendly alternatives, making sustainability accessible, practical, and engaging for all users.

At its core, Eco Cart allows users to search for commonly used items—such as plastic bottles or liquid detergents—and instantly discover greener substitutes like stainless steel flasks or detergent sheets. 

---

## ✨ Key Features

- 🔍 **Easy Discovery**: Instantly find eco-friendly product alternatives.
- 📊 **Impact Comparison**: View transparent metrics for **Carbon Footprint**, **Water Usage**, and **Plastic Waste**.
- 📈 **Interactive Dashboards**: Track your environmental savings through dynamic, real-time visualizations.
- 🎨 **Modern Interface**: A responsive, organic design built with smooth **Framer Motion** animations.
- 🤖 **AI Support**: Integrated **Botpress** chatbot for guided shopping and sustainability queries.

---

## 🛠️ Technology Stack

The project utilizes a bleeding-edge, high-performance tech stack:

### 🚀 Core Frameworks
- **Next.js (v16.2.2)**: Leveraging the latest **App Router** for superior performance.
- **React (v19.2.4)**: Utilizing modern hooks and concurrent rendering features.
- **TypeScript**: Ensuring enterprise-grade type safety and reliability.

### 🎨 Styling & Animation
- **Tailwind CSS (v4)**: Utilizing the next-generation CSS engine for ultra-fast styling.
- **Framer Motion**: Powering smooth, sophisticated UI transitions.
- **Custom SVGs**: Lightweight, high-quality iconography.

### 📉 Data & Effects
- **Chart.js & react-chartjs-2**: Fueling the real-time analytics in the Impact Dashboard.
- **canvas-confetti**: Adding celebratory feedback to reinforce sustainable habits.

### 💬 Integration & Deployment
- **Botpress Cloud**: Powering the intelligent webchat support system.
- **Vercel**: The preferred platform for seamless Next.js deployment.

## 🏁 Getting Started

1. **Clone the repo**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Open your browser**: [http://localhost:3000](http://localhost:3000)
